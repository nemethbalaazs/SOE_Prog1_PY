# HF01 – Alapok: kiíratás, változók, adattípusok

## Feladatok

### 1. feladat – Hello, World!

Írj egy Python programot, amely kiírja a képernyőre: `Hello, World!`

### 2. feladat – Névbevitel

Kérj be a felhasználótól egy nevet, majd köszöntsd személyre szabottan!

**Elvárt kimenet (példa):**
```
Adj meg egy nevet: Anna
Szia, Anna!
```

### 3. feladat – Adattípusok

Hozz létre az alábbi változókat és írd ki értéküket és típusukat (`type()` segítségével):
- egy egész szám (`int`)
- egy lebegőpontos szám (`float`)
- egy karakterlánc (`str`)
- egy logikai érték (`bool`)

### 4. feladat – Alapterület számítás

Kérj be egy téglalap szélességét és magasságát (lebegőpontos számként), majd számítsd ki és írd ki az alapterületét!

**Elvárt kimenet (példa):**
```
Szélesség: 5.5
Magasság: 3.0
Alapterület: 16.5
```

### 5. feladat – Hőmérséklet-átváltás

Kérj be egy hőmérsékletet Celsius-ban, majd számítsd ki Fahrenheit-ban is!

**Képlet:** `F = C * 9/5 + 32`

**Elvárt kimenet (példa):**
```
Hőmérséklet Celsius-ban: 100
100.0 °C = 212.0 °F
```
